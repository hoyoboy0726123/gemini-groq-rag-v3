# Gemini Groq RAG v3

A local Retrieval-Augmented Generation (RAG) application powered by Google Gemini and Groq.

## Features

- PDF upload & parsing
- Local vector search (IndexedDB)
- Chat with your documents via Gemini / Groq
- Export / Import conversations

## Live Demo

https://hoyoboy0726123.github.io/gemini-groq-rag-v3/

## Development

```bash
npm install
npm run dev
```
